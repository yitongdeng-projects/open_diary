Welcome to my open diary.

This is a static blog: every entry is just a Markdown file in a GitHub
repository. When I save a post, GitHub rebuilds the site automatically and
publishes it. No database, no server, nothing to maintain.

## Writing

I can write here directly in an editor, or from any device using the browser
editor. Either way, each post becomes a file I can re-edit anytime — and every
change is kept in the git history.
